Catatan API

Mini project REST API to serve Catatan App.



Resources

In order to be able to write, update and delete your notes, first you need to register as user.

/users
/notes


Routes

GET /users
GET /users/:username
POST /users/register
POST /users/login
PUT /users/:username
DELETE /users/:username

GET /notes
GET /notes/:id
POST /notes
PUT /notes/:id
DELETE /notes/:id

Note: See guide for usage example.


Stack used are Restify, MongoDB, JWT. Find me at github.
